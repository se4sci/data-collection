54         """
55         Compute electron-phonon coupling related quantities
56         as a function of temperature or at T=0. Those include:
57     
58             - the frequency-dependent self-energy
59     
60             - the spectral function
61     
62             - the quasiparticle renormalization
63               (the real part of the self-energy at the bare eigenvalues)
64     
65             - the quasiparticle broadening, or inverse lifetime
66               (the imaginary part of the self-energy at the bare eigenvalues)
67     
68     
69         Arguments (default values in parenthesis)
70         =========================================
71     
72     
73         Options
74         -------
75     
76         temperature: (True)
77             Compute the temperature dependence of all quantities.
78             Otherwise, they are computed only at T=0.
79     
80         renormalization: (True)
81             Compute the renormalization, that is,
82             the real part of the self-energy at the bare eigenvalues.
83     
84         broadening: (False)
85             Compute the broadening, that is, the inverse lifetime,
86             which is the imaginary part of the self-energy at the bare eigenvalues.
87     
88         self_energy: (False)
89             Compute the frequency-dependent self-energy,
90             with or without temperature dependence.
91     
92         spectral_function: (False)
93             Compute the spectral function.
94             Requires frequency-dependent self-energy (self_energy=True).
95     
96         double_grid: (False)
97             Activate the use of the double grid technique.
98             If True, the user must provide values for:
99             nqpt_fine, wtq_fine, eigq_fine_fnames, gkk_fine_fnames, ddb_fine_fnames
100     
101         dynamical: (True)
102             Use the dynamical AHC theory.
103             Otherwise, the static AHC theory is used.
104     
105         split_active: (True)
106             Split the active contribution from the sternheimer contribution.
107             This means that Abinit was run with ieig2rf=5
108             to produce both EIGR2D.nc and GKK.nc files.
109             If set to False, it means that Abinit was run with ieig2rf=1 
110             to produce only EIGR2D.nc files.
111     
112         mode: (False)
113             Do a mode-by-mode decomposition of the ZPR.
114     
115         write: (True)
116             Write the results on the disk.
117     
118         verbose: (False)
119             Print information to standard ouput as the calculation proceeds.
120     
121     
122         Parameters
123         ----------
124     
125         nqpt: (1)
126             Number of q-points.
127     
128         wtq: ([1.])
129             Weights of all the q-points.
130             Should sum up to 1, but will be normalized anyway.
131     
132         nqpt_fine: (1)
133             Number of q-points on the fine grid.
134     
135         wtq_fine: ([1.])
136             Weights of all the q-points on the fine grid.
137             Should sum up to 1, but will be normalized anyway.
138     
139         smearing_eV: (0.01)
140             Imaginary parameter (eta) used in the self-energy denominator.
141     
142         temp_range: ([0,300,50])
143             Minimum, maximum and step temperature for eigenvalues dependance.
144     
145         omega_range: ([0,0,1])
146             Minimum, maximum and step frequency for the self-energy.
147     
148     
149         File names
150         ----------
151     
152         rootname: ('epc.out')
153             Rootname for the output files.
154     
155         ddb_fnames: ([])
156             Names of _DDB files.
157     
158         eigk_fname: ('')
159             Name of the _EIG.nc file at k for the eigenvalues being corrected.
160     
161         eigq_fnames: ([])
162             Names of _EIG.nc files at k+q.
163     
164         eigr2d_fnames: ([])
165             Names of _EIGR2D.nc files.
166     
167         gkk_fnames: ([])
168             Names of _GKK.nc files.
169     
170         fan_fnames: ([])
171             Names of _FAN.nc files to use instead of GKK.nc files.
172             This option is maintained for backward compatibility.
173     
174         eigi2d_fnames: ([])
175             Names of _EIGI2D.nc files. Only relevant when split_active=False
176             and broadening=True.
177     
178         ddb_fine_fnames: ([])
179             Names of _DDB.nc files at k+q on the fine grid.
180     
181         eigq_fine_fnames: ([])
182             Names of _EIG.nc files at k+q on the fine grid.
183     
184         gkk_fine_fnames: ([])
185             Names of _GKK.nc files on the fine grid.
186     
187     
188         Returns
189         =======
190     
191         epc: EpcAnalyzer
192             Object containing the response function data
