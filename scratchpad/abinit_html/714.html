3     *   Simple ANSI c routine which returns cpu time in seconds.
4     *   Suitable for a call by a Fortran program.
5     *   Usage (fortran): call cclock(cpu) for double precision cpu.

16     void FC_FUNC(cclock,CCLOCK)

18
